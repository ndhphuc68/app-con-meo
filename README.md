## Run Locally

Clone the project

```bash
  git clone https://github.com/ndhphuc68/app-con-meo.git
```

Go to the project directory

```bash
  cd app-con-meo
```

Install dependencies

```bash
  npm install || yarn install
```

Run android

```bash
  npm run android || yarn android
```
